# CodeForces Solutions (Java)

**Build tool:** Maven &nbsp;|&nbsp; **Language:** Java &nbsp;|&nbsp; **Test framework:** JUnit 5

## Getting Started

### Prerequisites

- Java 8+
- Maven 3.6+

### Run all tests

```bash
mvn test
```

### Compile only

```bash
mvn compile
```
